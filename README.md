# Evaluación Parcial 2 - Landing Page: Yoga4Life

Este repositorio contiene la landing page responsive para la evaluación del segundo parcial.

* **Estudiante:** Ángel Steven Rodríguez Chávez
* **Tema Asignado:** Centro de Yoga (Tema N.º 15)
* **Nombre de la Landing Page:** Yoga4Life
* **Paleta de Colores:** Verde Menta (`#98FF98`), Lavanda (`#E6E6FA`) y Blanco (`#ffffff`)

## Estructura del Proyecto

* `index.html` - Página principal estructurada con HTML5 semántico y Bootstrap 5.3.
* `css/general.css` - Estilos generales de la página y declaración de la paleta en `:root`.
* `css/index.css` - Estilos específicos de las secciones y media queries responsive (mobile-first).
* `img/` - Carpeta de recursos gráficos y poses de yoga locales.
